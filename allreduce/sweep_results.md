# AllReduce Optimization Results (MI300X)

## Changes

| Item | Before | After |
|------|--------|-------|
| `effective_limit` (bytes<=256KB) | 16 | 64 |
| algo threshold (TP<=4) | 1stage if bytes<512KB | 1stage if bytes<64KB |
| algo threshold (TP<=8) | 1stage if bytes<=64KB | 1stage if bytes<64KB |

---
